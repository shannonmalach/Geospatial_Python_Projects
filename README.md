# Spatial Computing with Python: Geospatial Analysis Portfolio

## Overview

This repository branch contains a collection of projects completed as part of a Spatial Computing course. The projects demonstrate the application of Python-based geospatial analysis techniques using Google Colab, open-source geospatial libraries, public datasets, and web-based geographic data services.

The work focuses on developing reproducible geospatial workflows for data acquisition, spatial analysis, network modeling, and cartographic visualization. Collectively, these projects showcase practical experience in Geographic Information Systems (GIS), spatial data science, and geospatial software development.

Projects in this repository demonstrate methods for obtaining, processing, and managing large geospatial datasets from a variety of public sources.

* Spatial data cleaning and preparation
* Coordinate reference system management

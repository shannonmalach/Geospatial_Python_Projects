# Geographic Predictors of Property Crime in Washington, D.C.

## Project Overview

This project investigates the spatial relationship between urban infrastructure and property crime in Washington, D.C. Using Geographic Information Systems (GIS), spatial statistics, and regression analysis, the study examines whether geographic features such as public transportation access, liquor license locations, and homeless shelter locations are significant predictors of property crime patterns across the city.

The project was completed as part of coursework in Spatial Computing and demonstrates the application of geospatial analysis techniques to a real-world public safety issue.

---

## Research Question

**To what extent do geographic and urban socioeconomic factors influence the spatial distribution of property crime in Washington, D.C.?**

The analysis evaluates the relationship between property crime incidents and several explanatory variables, including:

* Homeless shelter locations
* Liquor license locations
* Metro station locations
* Metrobus stop locations

Property crime categories analyzed include:

* Burglary
* Motor Vehicle Theft
* Theft from Auto
* Robbery

---

## Methodology

The project follows a spatial analytical workflow consisting of:

1. Data Collection and Preparation

   * Acquisition of crime and infrastructure datasets
   * Data cleaning and standardization
   * Coordinate system management and spatial joins

2. Exploratory Spatial Data Analysis

   * Descriptive mapping
   * Spatial visualization of crime distributions
   * Hot spot identification

3. Spatial Statistical Analysis

   * Global Moran's I
   * Local Moran's I (LISA)
   * Cluster and outlier analysis

4. Regression Modeling

   * Ordinary Least Squares (OLS) regression
   * Assessment of relationships between property crime and explanatory variables
   * Evaluation of model performance and statistical significance

## Results

The analysis identified statistically significant spatial patterns in property crime distribution across Washington, D.C. Spatial autocorrelation measures indicated that crime events were not randomly distributed and exhibited clustering in specific geographic areas. Regression analysis was used to evaluate the extent to which selected urban infrastructure variables contributed to observed crime patterns.

Detailed findings, model outputs, and interpretation are available in the accompanying research paper.
